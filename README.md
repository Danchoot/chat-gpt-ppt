## Showcase

1. Some topics for presentation named `topic.txt`:

```
what's OpenAI?
how OpenAI works?
what is the future of OpenAI?
```

2. Save your openapi token to `token.txt`.
3. Generate a ppt in seconds:

```
./cgp
```
