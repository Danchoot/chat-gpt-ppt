## Show case

Some topics for presentation named `topics.txt`:

```
what's OpenAI?
how OpenAI works?
what is the future of OpenAI?
```

Easily generate a simple ppt in seconds:

```
./cgp token.txt topics.txt output.ppt
```

And you get one:
